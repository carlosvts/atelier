# OpenCV

The first study environment of **atelier**.

This project contains small experiments exploring computer vision with OpenCV and NumPy. The goal is to build a solid understanding of the fundamentals before moving on to more advanced topics.

## Goals

* Learn the OpenCV API.
* Understand how images are represented and processed.
* Explore computer vision concepts through hands-on experiments.
* Build a foundation for future projects involving robotics and AI.

## Structure

```text
.
├── assets/
├── experiments/
├── main.py
├── pyproject.toml
└── README.md
```

## Running

Install the dependencies:

```bash
uv sync
```

Run the project:

```bash
uv run main.py
```
